I went with the Milligram framework (Links to an external site.), it's a ultra light-weight framework designed for better performance and higher productivity for clean code. It has a minimalist approach to make your code clutter-free. I think it's a great starting point for beginners as this framework isn't overwhelming with a bunch of components, but that's also it's downfall as frameworks like bootstrap have more resources and more styling components. I believe it's still a relatively new framework so there isn't much documentation and other resources like the other big players in frameworks.
