Interesting video this week, I always thought that as a web developer, we did everything; Such as designing and coding a website. But it looks like web developer is a very broad term. Web designers only design basic brochure pages, UI/UX designer only design the user interaction functionality and they don't even necessarily have to be able to code the functionality of the webpage. Wiki, (Links to an external site.) "The goal of user interface design is to make the user's interaction as simple and efficient as possible, in terms of accomplishing user goals" talks about how the goal is to maximize efficiency for user goals; to me this isn't what I really want to do. Wiki, (Links to an external site.) "User experience design (UX, UXD, UED or XD) is the process of enhancing user satisfaction by improving the usability, accessibility, and pleasure provided in the interaction between the user and the product." talks about UX enhancing user satisfaction; I think this is more up my alley, it's always interesting to me when I visit a website and they have very cool animations or designs to make their products really pop out of the page; I always wonder how they manage to come up with that particular design as I try to reverse engineer their page in my head.
