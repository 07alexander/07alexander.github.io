My background knowledge with Chrome Developer tools is pretty small. I never knew how much power the Developer Tools had when I first played with them. The video tutorial taught me how easy it is to manipulate a web page on-the-fly. It even has a live preview of what you do on the page, you don't have to refresh the page every time you change code. All of the things covered in the video is all new to me, I just really had no idea how powerful the chrome developer tools really are. I always thought that I had to download the html file then manipulate it myself and open it on a local environment. In the screenshot I provided, all I did was manipulate this page on canvas to display some different texts than the original. I now have a pretty good understanding of what chrome developer tools are and how powerful they can be.
