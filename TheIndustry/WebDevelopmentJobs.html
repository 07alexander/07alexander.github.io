Web Development or Web Related Jobs

Chrome Developer Summit 2016 / Dark Web / FTP Software / Becoming a Web Developer: Full Stack vs Front End / Web Development or Web related jobs

Using keywords, web developer jobs in Fresno California, in google search engine I found a website called indeed (Links to an external site.). Looking through indeed, it confirmed what I thought about Fresno's opportunity for web development careers, offering only 11 jobs offers. So I expanded my scope to all of California (Links to an external site.), giving me a total of 9398 jobs available for hire. While Fresno still has a lot more growing to do, California is booming with job opportunities for web developers with an average of $92,202 per year salary, that doesn't sound too bad. From the low-end to the high-end, every company wants HTML and CSS experience at least and most needing JavaScript and SQL. Most job offers also require a B.S. with most asking for a computer science degree. Even the Jr. Web developer job require a lot of experience, I honestly didn't think they required so much experience for entry level positions. I really have to step up my game and apply myself.
